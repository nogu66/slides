# まだ間に合う！Claude Code元年をふりかえる

2025年にリリースされたClaude Codeを振り返り、SubAgentとAgent Skillsの使い分けについて紹介するプレゼンテーションです。

## セットアップ

### 1. 依存関係のインストール

```bash
npm install
```

### 2. 開発サーバーの起動

```bash
npm run dev
```

ブラウザが自動的に開き、プレゼンテーションが表示されます。

## スクリプト

- `npm run dev` - 開発サーバーを起動（ホットリロード付き）
- `npm run build` - 本番用にビルド
- `npm run export` - PDFにエクスポート

## デザイン

このプレゼンテーションは以下の配色を使用しています：

- **背景色**: `#FAF9F5` (柔らかいベージュ)
- **アクセントカラー**: `#D97757` (温かみのあるオレンジ)
- **文字色**: `#000000` (黒)
